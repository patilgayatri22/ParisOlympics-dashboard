
# Paris Olympics Dashboard

An interactive data analysis and visualization project focused on the **Paris Olympic Games**.  
This repository includes a Power BI dashboard (`*.pbix`) and a Jupyter notebook for exploratory data analysis of the Paris Olympics dataset.


##  Overview

This project aims to provide insights and visualizations into the **Paris Olympic Games** using data analytics and dashboarding tools. It enables users to explore medal distributions, country performance, athlete data, and more through visually engaging charts and interactive elements.

Key components:
- **Power BI Dashboard** (`Paris_Olympic_Project.pbix`)  
- **Exploratory Data Analysis Notebook** (`paris-eda.ipynb`)  
- **Raw or prepared data** in the `data/` folder  
- **MIT License** included
